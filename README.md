# Student Grades API

простой REST-сервис на FastAPI для загрузки и анализа успеваемости студентов.

## Возможности

- Загрузка CSV файлов с оценками студентов
- Валидация входных данных
- Аналитика успеваемости через SQL запросы
- Работа без ORM (только SQL через asyncpg)
- Docker Compose для быстрого развертывания


### Запуск
```bash
# Запуск сервисов
docker-compose up -d

# Проверка логов
docker-compose logs -f api
```


### API Endpoints

## POST /upload-grades
Загрузка CSV файла с оценками.

**Формат CSV:**
```csv
Дата;Номер группы;ФИО;Оценка
11.03.2025;101Б;Курочкин Антон Владимирович;4
18.09.2024;102Б;Москвичев Андрей;3
26.09.2024;103М;Фомин Глеб Александрович;4
```

**Ответ:**
```json
{
  "status": "ok",
  "records_loaded": 2000,
  "students": 40
}
```

### GET /students/more-than-3-twos
Студенты с более чем 3 двойками.

**Ответ:**
```json
[
  { "full_name": "Иванов Иван", "count_twos": 5 }
]
```

### GET /students/less-than-5-twos
Студенты с менее чем 5 двойками.

**Ответ:**
```json
[
  { "full_name": "Петров Пётр", "count_twos": 2 }
]
```
